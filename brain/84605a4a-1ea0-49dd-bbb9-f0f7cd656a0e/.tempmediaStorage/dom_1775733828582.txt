The Leave Tracker system is a distributed application running across three specialized Windows Server Virtual Machines. Maintaining visibility across this stack is critical for ensuring zero-downtime availability for employee leave management.

Self-Healing Watchdogs
Before manual intervention is required, the system employs primary self-healing scripts that run every 5 minutes on VM1 and VM2. These should be monitored as "Last-Line-of-Defense" tasks.
